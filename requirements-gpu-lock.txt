# GPU-enabled PyTorch lock for the NeuroMultiverse analysis environment.
#
# Separate from requirements-lock.txt because the correct PyTorch wheel depends
# on the local NVIDIA driver and is not resolvable from PyPI. Installing from
# PyPI yields the CPU-only build, which fails the GPU gate.
#
# Every version below was read from the environment after a real installation
# with `pip show` and `pip list --format=freeze`. They are measurements, not
# estimates, and no transitive version here was hand-authored.
#
# ---------------------------------------------------------------------------
# Verified installation
# ---------------------------------------------------------------------------
# Verification date:            2026-07-17
# Authoritative wheel index:    https://download.pytorch.org/whl/cu130
# Python:                       3.11.9 (CPython, Windows x86_64)
# pip:                          26.1.2
# Operating system:             Windows 11 Home, build 26200, 64-bit
# NVIDIA driver (nvidia-smi):   610.47
# CUDA UMD reported by driver:  13.3
# torch CUDA build:             13.0
# Verified GPU:                 NVIDIA GeForce RTX 4070 Laptop GPU
# torch.cuda.is_available():    True
# torch.cuda.device_count():    1
#
# ---------------------------------------------------------------------------
# Exact reinstall command
# ---------------------------------------------------------------------------
#     python -m pip install torch==2.13.0+cu130 \
#         --index-url https://download.pytorch.org/whl/cu130
#
# The `--index-url` flag is mandatory. A bare `pip install torch` resolves
# against PyPI and installs a CPU-only build that reports cuda_available False.
#
# ---------------------------------------------------------------------------
# Reverification requirement
# ---------------------------------------------------------------------------
# Do NOT treat this file as a permanent fact. Before any future rebuild,
# reverify against the official PyTorch installation source
# (https://pytorch.org/get-started/locally/, cross-checked against
# https://download.pytorch.org/whl/):
#
#   * that the cu130 index still publishes a wheel for the target Python;
#   * that the local NVIDIA driver still satisfies the CUDA build's minimum;
#   * that the pinned torch version is still available at that index.
#
# Wheel availability and CUDA support windows change over time, and a driver
# upgrade or downgrade can invalidate this pin. Re-read the official command
# at install time rather than copying the one above blindly.
#
# ---------------------------------------------------------------------------
# Pinned packages
# ---------------------------------------------------------------------------
# Only what is required to reproduce the GPU-enabled torch installation.
# torchvision is deliberately absent: this project has no computer-vision
# requirement, no tracked source imports it, and it pulled in Pillow for no
# reason. Both were uninstalled on 2026-07-17.
